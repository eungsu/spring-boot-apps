#  프론트엔드 애플리케이션
회원가입, 로그인, 게시글, 댓글 관리 기능을 지원하는 react기반의 프론트 엔드 애플리케이션이다.

## 실행방법
1. `spring-boot-apps`를 내려받는다.
   ```bash
   git clone https://github.com/eungsu/spring-boot-apps.git
   ```
2. `app1-backend` 애플리케이션을 실행시킨다.
   ```bash
   cd spring-boot-apps
   cd app2-backend
   
   nvnw  spring-boot:run
   ```
